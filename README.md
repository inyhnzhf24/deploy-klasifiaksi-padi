Klasifikasi Penyakit Daun Padi
Aplikasi ini menggunakan model Convolutional Neural Network (CNN) untuk mengklasifikasikan penyakit pada daun padi berdasarkan citra. 
Model dapat mengenali 4 kategori: Bacterial Blight, Blast, Brown Spot, dan Tungro.
